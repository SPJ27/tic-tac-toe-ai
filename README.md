# tic-tac-toe ai

## What is the project?
it is a basic tic tac toe game that users can play against computer. it uses minimax algorithm to figure out the best move to play by calculating the highest score free each tree.

## Why I Built it?
I built it so that I can get back to python programming and make games, and even though it is is a very basic terminal app, it is fun to play.

## How to test?
From the releases, download main.zip and extract it. Run, the main.exe to run the game.

## Why the project fits the theme?
This tic-tac-toe game fits the theme because it is a playable game that runs in your terminal.

## Inspiration?
I read about how bill gates created a similar game during his high school, and decided it would be good game to get baclk to python programming.
